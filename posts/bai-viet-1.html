Tối Ưu Hóa Giao Diện Lịch với CSS Grid/Flexbox

Đăng ngày: 26/11/2025 | Tác giả: Dev Team

Xây dựng giao diện lịch 7 cột đáp ứng (responsive) là một thách thức phổ biến trong phát triển frontend. Bài viết này sẽ hướng dẫn bạn sử dụng CSS Grid và Flexbox một cách hiệu quả.

I. Tại sao nên dùng CSS Grid?

CSS Grid là công cụ mạnh mẽ nhất để tạo bố cục dạng lưới (grid layout). So với Flexbox, Grid cho phép kiểm soát cả 2 chiều (hàng và cột) một cách dễ dàng.

Cấu trúc HTML cơ bản

<div class="calendar-grid">
  <div class="day-header">T2</div>
  <div class="day-header">T3</div>
  <!-- ... 7 cột -->
  
  <div class="day-cell">1</div>
  <div class="day-cell">2</div>
  <!-- ... các ngày -->
</div>

CSS Grid cho Lịch 7 cột

.calendar-grid {
  display: grid;
  grid-template-columns: repeat(7, 1fr);
  gap: 8px;
  padding: 16px;
}

.day-cell {
  aspect-ratio: 1;
  display: flex;
  align-items: center;
  justify-content: center;
  border-radius: 8px;
  transition: background-color 0.2s;
}

.day-cell:hover {
  background-color: var(--hover-bg);
  cursor: pointer;
}

/* Responsive cho mobile */
@media (max-width: 768px) {
  .calendar-grid {
    gap: 4px;
    padding: 8px;
  }
}

II. Kết hợp Flexbox cho các thành phần nhỏ

Trong khi Grid phù hợp cho bố cục tổng thể, Flexbox lại xuất sắc trong việc căn chỉnh nội dung bên trong mỗi ô lịch.

.day-cell-content {
  display: flex;
  flex-direction: column;
  gap: 4px;
}

.lunar-date {
  font-size: 0.75em;
  color: var(--text-muted);
}

III. Tối ưu hiệu suất

Một vài tips để giao diện lịch chạy mượt mà:

  • Sử dụng will-change: transform cho các animation
  • Tránh dùng box-shadow phức tạp, thay bằng border
  • Lazy load các tháng trước/sau khi cần
