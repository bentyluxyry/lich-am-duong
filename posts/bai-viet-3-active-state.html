✨ Kỹ Thuật Hiển Thị Trạng Thái Ngày Hiện Tại (Active State)

Đăng ngày 26/11/2025 | Chuyên mục: JS & UI/UX

Việc làm nổi bật ngày hiện tại hoặc ngày mà người dùng đang chọn là một phần quan trọng trong bất kỳ ứng dụng lịch nào. Đây là cách Vạn Niên Số sử dụng kết hợp JavaScript (quản lý logic) và CSS (quản lý giao diện) để tạo ra hiệu ứng nổi bật với màu Tím.

1. Định nghĩa Class CSS cho Ngày Hôm Nay (`.today`)

Trong Vạn Niên Số, chúng ta sử dụng class .today để làm nổi bật ngày hiện tại, đảm bảo nó luôn được nhìn thấy rõ ràng trên lịch.


/* Đã có trong style.css */
.calendar-day.today { 
    background-color: var(--primary-color); 
    color: white !important; 
    border-radius: 50%; /* Tròn hoàn toàn */
    font-weight: 700;
    box-shadow: 0 2px 8px rgba(0, 0, 0, 0.2);
    transform: scale(1.05); /* Hiệu ứng nổi bật */
    transition: transform 0.3s ease-out; /* Thêm transition */
}

2. Logic Xác định Ngày Hiện Tại trong JavaScript

Hàm renderCalendar() trong `main.js` chịu trách nhiệm so sánh ngày đang vẽ với ngày hiện tại của hệ thống và thêm class `today` tương ứng.


// Trong hàm renderCalendar()...
today = new Date(); // Cập nhật lại ngày hôm nay
for (let i = 1; i <= daysInMonth; i++) {
    // ...
    const dayDiv = document.createElement('div');
    // ...
    
    // Đánh dấu ngày hôm nay (today)
    if (i === today.getDate() && currentMonth === today.getMonth() && currentYear === today.getFullYear()) {
        dayDiv.classList.add('today');
    }
    
    // ...
    DOM.calendarGrid.appendChild(dayDiv);
}

Lưu ý về Trạng thái Active cho Ngày được Chọn

Trong thiết kế này, chúng ta không dùng một class `.active` riêng cho ngày được chọn click vào. Thay vào đó, việc click chỉ mở Modal Chi Tiết Ngày. Class `.today` được dùng riêng biệt và cố định cho ngày hệ thống.
