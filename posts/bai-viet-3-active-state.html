Kỹ Thuật Hiển Thị Trạng Thái Ngày Hiện Tại (Active State)

Đăng ngày: 26/11/2025 | Tác giả: Dev Team

Trạng thái **Active State** hay **Ngày được chọn** (màu tím) là một yếu tố quan trọng trong UI/UX của lịch. Nó giúp người dùng biết họ đang tập trung vào ngày nào.

I. CSS cho Active State

Chúng ta sử dụng một class CSS đơn giản là .active và áp dụng các kiểu dáng nổi bật (màu nền, bo tròn) cho nó.

Định nghĩa Style Active

.day-cell.active {
    background-color: var(--active-day-bg); /* Màu tím */
    color: var(--active-day-text); /* Màu trắng */
    border-radius: 50%; /* Hình tròn */
    width: 45px;
    height: 45px;
    box-shadow: 0 4px 8px rgba(121, 82, 179, 0.4); /* Thêm bóng nổi */
    transition: all 0.3s ease; /* Chuyển động mượt mà */
}

II. JavaScript cho Chuyển đổi Trạng thái

Logic trong main.js đảm nhiệm việc quản lý trạng thái này:

  1. **Khởi tạo:** Khi lịch được tải (renderCalendar), kiểm tra và thêm class .active vào ngày hôm nay.
  2. **Tương tác (Click):** Khi người dùng nhấp vào một ô ngày khác, cần thực hiện hai bước sau:
    • Xóa class .active khỏi tất cả các ô ngày.
    • Thêm class .active vào ô ngày vừa được nhấp.

Mã JavaScript xử lý Click

dayCell.addEventListener('click', function() {
    // 1. Xóa trạng thái active cũ
    document.querySelectorAll('.day-cell').forEach(cell => {
        cell.classList.remove('active');
    });

    // 2. Thêm trạng thái active mới
    this.classList.add('active');
});

Nhờ có thuộc tính transition: all 0.3s ease; trong CSS, việc thêm và xóa class .active sẽ tạo ra hiệu ứng chuyển đổi mượt mà thay vì thay đổi đột ngột.
