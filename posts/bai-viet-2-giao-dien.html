Tối Ưu Hóa Giao Diện Lịch với CSS Grid/Flexbox

Đăng ngày: 26/11/2025 | Tác giả: Dev Team

Để tạo ra bố cục 7 cột chuẩn mực của lịch, **CSS Grid** là lựa chọn vượt trội so với các phương pháp cũ. Nó cho phép định nghĩa một hệ thống lưới hai chiều (hàng và cột) một cách trực quan.

I. Xây dựng Lưới (Grid System)

Kỹ thuật cốt lõi là sử dụng thuộc tính grid-template-columns với đơn vị fr (fractional unit) để đảm bảo 7 cột luôn có chiều rộng bằng nhau và đáp ứng (responsive) hoàn hảo.

Mã CSS cho Lưới Lịch

.calendar-grid {
    display: grid;
    /* Chia 7 cột bằng nhau, chiếm toàn bộ không gian có sẵn */
    grid-template-columns: repeat(7, 1fr); 
    text-align: center;
}

.day-cell {
    /* Sử dụng Flexbox để căn giữa nội dung Ngày và Âm Lịch */
    display: flex;
    flex-direction: column;
    justify-content: center;
    align-items: center;
    height: 60px; 
}

II. Tối ưu Responsive (Mobile First)

Ứng dụng được thiết kế theo tư duy **Mobile First**, nghĩa là CSS mặc định là dành cho mobile. Media Query chỉ được dùng để **tăng** kích thước hoặc **thay đổi** bố cục khi màn hình đủ lớn.

Tùy chỉnh cho Desktop (Media Query)

Ví dụ, tăng chiều cao ô ngày để tận dụng không gian trên máy tính:

@media (min-width: 768px) {
    .calendar-card {
        padding: 25px;
    }
    .day-cell {
        height: 70px; /* Tăng chiều cao trên desktop */
    }
}
