📐 Tối Ưu Hóa Giao Diện Lịch với CSS Grid/Flexbox

Đăng ngày 26/11/2025 | Chuyên mục: Frontend

Việc tạo ra bố cục cho lịch là một ví dụ hoàn hảo để áp dụng **CSS Grid**. Khác với việc sử dụng các thuộc tính cũ như float hoặc thậm chí cả **Flexbox** (khi cần cả hàng và cột), Grid giúp ta định nghĩa cấu trúc 2 chiều một cách rõ ràng và mạnh mẽ hơn.

1. Thiết lập Cấu trúc HTML

Chúng ta cần một thẻ cha bao bọc .calendar-grid và các thẻ con .calendar-header-day (nhãn thứ) cùng với các ô ngày .calendar-day.


<div class="calendar-grid">
    <div class="calendar-header-day">CN</div>
    ...
    <div class="calendar-header-day">T7</div>

    <!-- Ngày tháng sẽ được JS chèn vào -->
    <div class="calendar-day">...</div>
    <div class="calendar-day today">...</div>
    ...
</div>

2. Áp dụng CSS Grid cho Bố cục 7 Cột

Thuộc tính then chốt là display: grid; và grid-template-columns. Chúng ta sử dụng hàm repeat(7, 1fr) để tạo ra 7 cột có chiều rộng bằng nhau (1 phần - 1 fraction), đảm bảo tính **responsive** tuyệt đối.

Mã CSS Cốt lõi:


.calendar-grid {
    display: grid;
    /* Tạo 7 cột có chiều rộng bằng nhau. */
    grid-template-columns: repeat(7, 1fr); 
    gap: 2px; /* Khoảng cách giữa các ô */
    text-align: center;
}

.calendar-day { 
    display: flex; 
    flex-direction: column; 
    justify-content: center; 
    align-items: center; 
    /* Dùng aspect-ratio để giữ ô luôn là hình vuông */
    aspect-ratio: 1 / 1; 
    /* ... các style khác ... */
}

Bằng cách này, dù màn hình di động nhỏ hay desktop lớn, bố cục lịch luôn duy trì được 7 cột hoàn hảo và các ô ngày tự động căn chỉnh để giữ tỷ lệ hình vuông, tạo giao diện chuyên nghiệp và rõ ràng.
