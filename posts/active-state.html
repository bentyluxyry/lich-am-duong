✨ Kỹ Thuật Hiển Thị Trạng Thái Ngày Hiện Tại (Active State)

Đăng ngày 26/11/2025 | Chuyên mục: JS & UI/UX

Việc làm nổi bật ngày hiện tại hoặc ngày mà người dùng đang chọn là một phần quan trọng trong bất kỳ ứng dụng lịch nào. Đây là cách Vạn Niên Số sử dụng kết hợp JavaScript (quản lý logic) và CSS (quản lý giao diện) để tạo ra hiệu ứng nổi bật với màu Tím.

1. Định nghĩa Class CSS `active`

Chúng ta cần một class CSS duy nhất, ví dụ là .active, để áp dụng tất cả các hiệu ứng nổi bật (màu nền, màu chữ, bo tròn, v.v.).


.calendar-day.active {
    background-color: #6C5CE7; /* Màu tím bạn thấy trong ảnh */
    color: white;
    font-weight: bold;
    /* Dùng border-radius: 50% để tạo hình tròn */
    border-radius: 50%; 
    /* Thêm hiệu ứng chuyển đổi mượt mà */
    transition: all 0.2s ease-in-out;
}

2. Quản lý Trạng thái bằng JavaScript

JavaScript chịu trách nhiệm tìm ra ô ngày nào cần được làm nổi bật và thêm/bỏ class .active tương ứng. Có hai trường hợp chính:

2.1. Hiển thị Ngày Hiện Tại (Initial Load)

Khi trang được tải lần đầu, chúng ta sử dụng JavaScript để tìm ngày hiện tại dựa trên thuộc tính dữ liệu (ví dụ: data-date) và áp dụng class.


document.addEventListener('DOMContentLoaded', () => {
    // 1. Lấy ngày hiện tại ở định dạng YYYY-MM-DD
    const today = new Date().toISOString().slice(0, 10); // Ví dụ: "2025-11-26"
    
    // 2. Tìm phần tử tương ứng
    const activeDay = document.querySelector(`.calendar-day[data-date="${today}"]`);

    // 3. Thêm class 'active'
    if (activeDay) {
        activeDay.classList.add('active');
    }
});

2.2. Xử lý Sự kiện Click (User Interaction)

Khi người dùng nhấp vào một ngày khác, chúng ta phải loại bỏ class .active khỏi ngày cũ và áp dụng nó cho ngày mới.


document.querySelectorAll('.calendar-day').forEach(dayElement => {
    dayElement.addEventListener('click', (event) => {
        // 1. Tìm và loại bỏ class 'active' khỏi phần tử cũ
        const oldActive = document.querySelector('.calendar-day.active');
        if (oldActive) {
            oldActive.classList.remove('active');
        }
        
        // 2. Thêm class 'active' vào phần tử vừa click
        event.currentTarget.classList.add('active');
    });
});
